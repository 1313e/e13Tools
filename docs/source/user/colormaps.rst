Colormaps
=========
.. deprecated:: 0.6.16
    All colormaps in *e13Tools* have been ported to the `CMasher`_ package.
    See `its documentation`_ on colormaps for an overview of all colormaps that used to be here, and possibly more.
    All colormaps are still available by installing and importing *e13Tools* as before, but this backwards compatibility will be removed in v0.7.0.
