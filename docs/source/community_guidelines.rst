Community guidelines
====================
*e13Tools* is an open-source and free-to-use software package (and it always will be), provided under the BSD-3 license (see below for the full license).

Users are highly encouraged to make contributions to the package or request new features by opening a `GitHub issue`_.
If you would like to contribute to the package, but do not know what, then there are quite a few ToDos in the code that may give you some inspiration.
As with contributions, if you find a problem or issue with *e13Tools*, please do not hesitate to open a `GitHub issue`_ about it.

.. _GitHub issue: https://github.com/1313e/e13Tools/issues
